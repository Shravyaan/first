# HTML Concepts

- Paired Tags
- Unpaired Tags (Self closing tags)
- Block level tags (div, p, ul, ol, section) entire block execution
- Inline tags (img, a, b, i)
- Semantic tags (header, nav, section, footer)
- Non-Semantic tags (div, span)

1. **Paired tags**:
   A paired tags are the tags which needs to be closed properly after opening.
   Ex: `<h1></h1>, <p></p>, <html></html>`
